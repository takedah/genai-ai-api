# 源内への登録パラメータ

デプロイ後、源内にAPIを登録する際に使用するパラメータです。

## 名前
法令レポート生成

## 概要
### 想定用途
法令に関する質問に対して、AIが意図を解析し、最適化された構造のレポートを動的に生成します。法的調査、政策立案支援、法令解釈の確認などに役立ちます。出力結果はAIが生成した内容ですので、正確性については必ず原典を確認してください。

:::disclosure{title=アプリ個別の留意点}
BigQuery MLのベクトル検索を使用しているため、データの更新頻度は手動更新に依存します。最新の法令改正情報については、e-Gov法令検索で確認することをお勧めします。
:::

:::disclosure{title=入力例（初利用時は要チェック）}
"デジタル社会形成基本法における「デジタル社会」の定義を教えてください"
"個人情報保護法と行政機関個人情報保護法の違いを比較してください"
:::

## 使い方
### 操作方法
法令に関する質問を自由に入力してください。システムが自動的に質問の種類を判定し、最適な構造のレポートを生成します。

:::disclosure{title=対応する質問パターン}
1. **定義確認型**: 法令用語の定義を知りたい場合
2. **手続き確認型**: 具体的な手続きの流れを知りたい場合
3. **比較検討型**: 複数の法令や制度を比較したい場合
4. **解釈適用型**: 特定の条文の解釈や適用を知りたい場合
5. **政策研究型**: 政策的な分析や提言が欲しい場合
6. **包括分析型**: 広範囲な法制度の全体像を知りたい場合
:::
